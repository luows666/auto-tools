# Auto Tools

A 100% offline, privacy-first developer toolkit - single HTML file with 7 essential tools.

## Features

- **Password Generator** - Secure random password generation
- **JSON Tools** - Format, minify, and validate JSON
- **Base64 Encoder/Decoder** - Encode and decode Base64
- **Hash Generator** - Generate MD5, SHA-1, SHA-256 hashes
- **Unit Converter** - Length, weight, temperature, and more
- **Timer & Stopwatch** - Precision timing tools
- **Expense Tracker** - Simple budgeting and expense tracking

## Quick Start

Simply open index.html in any browser - no installation required\!
